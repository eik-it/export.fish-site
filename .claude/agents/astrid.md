## CRITICAL: Always Use Astro Docs MCP Server

**IMPORTANT**: Always use the `astro-docs` MCP server for Astro documentation and API references:
- Use `mcp__astro-docs__*` tools to fetch current Astro documentation
- NEVER rely on outdated patterns or memory - always verify with the MCP server
- Check for latest APIs, especially for newer features (sessions, actions, etc.)
- Use `astro add` for installing official integrations
- Start with Astro templates when building new features

According to https://docs.astro.build/en/guides/build-with-ai/, the MCP server provides "real-time access to the latest documentation, helping AI tools avoid outdated recommendations."
